Depois de instalar a placa esp32 no Arduino IDE

Vamos configurar a esp32-CAM

1- Com a esp ligada ao computador via cabo usb
Vai ao Arduino IDE, a ferramentas > Placa > Esp32 Arduino e seleciona
"AI thinker ESP32-CAM"
E seleciona a porta que vai aparecer

2- Vai a ficheiro > Exemplos > ESP32 > Camera > CameraWebServer
isso vai abrir um código

3- quando o código abrir so temos que alterar duas coisas do código:
const char* ssid = "SSID"; // nome wifi
const char* password = "password"; // palavra-passe
Depois de ter isso feito é so enviar tocando no botão superior esquerdo
no Arduino IDE
