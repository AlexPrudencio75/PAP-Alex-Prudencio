Shield

Uma shield é uma placa de expansão especificamente para Arduino NANO é projetada para permitir o fácil acesso dos pinos de interface de um Arduino Nano. Cada pino DIO ou pino analógico da Nano é trazido para um pino conveniente, que também permite a conexão de servos digitais. Esta placa também inclui um soquete DC e circuito regulador, de modo que o Nano pode ser alimentado a partir de uma fonte de alimentação externa.

Arduino Nano I / O Extension Shield (Suporta Nano 3.0)

- 2 tiras de pinos fêmeas para conectar Nano 3.0 PCB

- Cada I/O tem 3 pinos machos: GND, Tensão e Sinal I/O ~

- 1 regulador de amplificador para alimentar NANO e
outros circuitos

- Pinos A0 - A7 e D0 - D13 disponíveis em pinos machos

- Pinos para GND e 3.3V

- Pinos para RX, TX, 5V, GND


- Pinos para SCL, SDA, 5V e GND


- LED de alimentação


- Interruptor de reposição


- Dimensões da PCB: 60mm x 55mm.
